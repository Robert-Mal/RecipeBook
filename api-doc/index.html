Recipe Book GraphQL API

Web app for creating recipes, eating plans and shopping lists

API Endpoints
http://localhost:3000/graphql

Queries

eatingPlan

Response

Returns an EatingPlan!

Arguments
Name Description
id - String!

Example

Query
query eatingPlan($id: String!) {
  eatingPlan(id: $id) {
    _id
    name
    description
    meals {
      ...MealFragment
    }
    isPublic
    createdAt
    username
  }
}
